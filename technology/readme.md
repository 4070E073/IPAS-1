# 網管工程師升級==>SSCP®資安專業人員
```
SSCP®資安專業人員認證(Systems Security Certified Practitioner)是一張針對企業組織中負責網路及系統管理的相關人員，
擁有資安技術能力且具實務經驗者所設立的證照。
取得SSCP®認證，可確認受測者在資安技術領域中具有一定的專業實務能力。

https://www.uuu.com.tw/Course/Show/46/SSCP資安專業人員認證課程
```
